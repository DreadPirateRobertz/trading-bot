# Trading Bot

Aggressive crypto/traditional market trading bot with social media sentiment analysis.

## Strategy
- Monitor crypto and traditional markets in real-time
- Crawl Reddit (r/wallstreetbets, r/CryptoCurrency, etc.) for sentiment
- Observe social media and news trends across all channels
- Make quick, advantageous trading decisions
- High risk / high reward approach
- Eventually autonomous trade execution

## Architecture (TBD)
- Market data APIs (crypto exchanges, stock market feeds)
- Social sentiment pipeline (Reddit, Twitter/X, news)
- Signal generation and risk management
- Trading execution via Robinhood/Alpaca/exchange APIs
- Dashboard for monitoring and manual override
